# Crypto Liquidity Manager

Python-based liquidity and inventory monitoring tool for cryptocurrency centralized exchanges (CEXs).

## Features

* Real-time order book monitoring
* Spread analysis
* Inventory risk tracking
* Threshold-based liquidity alerts
* Binance API integration using CCXT

## Technologies Used

* Python
* CCXT
* Pandas
* Logging

## Future Improvements

* Multi-exchange support
* WebSocket integration
* Automated hedging logic
* Dashboard visualization
